# Global-YouTube-Analytics
Analyzing subscriber growth, video views, earnings, and creator performance trends using Python and Power BI.

## Project Overview

This project analyzes global YouTube channel statistics to uncover insights related to subscribers, video views, earnings, content categories, and country-wise creator distribution.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Power BI
* Google Colab

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis (Q1–Q20)
3. Data Visualization
4. Dashboard Creation using Power BI
5. Insight Generation

## Key Insights

* Entertainment and Music categories dominate YouTube.
* Subscriber count is strongly related to video views.
* Earnings vary significantly across categories.
* Creator activity is concentrated in a few countries.
* Audience engagement influences creator success.
